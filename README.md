# attendance-app

Attendance Mobile Application built in Clojure

## Usage

```
re-natal use-figwheel && react-native run-android
```

## Development

```
lein repl
(start-figwheel! "android")
```

## REPL

```
(in-ns 'attendance-app.android.core)
```
